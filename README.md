# AntiWhiteList

Инструмент для обхода блокировок на Windows. Позволяет использовать интернет на ПК и телефоне через зашифрованный туннель, без VPS и платных сервисов.

## Что внутри

| Компонент | Описание |
|-----------|----------|
| **Xray** (VLESS+Reality) | Прокси-сервер, маскируется под обычный HTTPS |
| **zapret** | Обход DPI провайдера |
| **CloudPub** (`clo.exe`) | Бесплатный TCP туннель — пробрасывает порты без проброса портов на роутере |
| **Meshnet** | Веб-дашборд для управления туннелями и настройки телефона |

## Требования

- Windows 10 / 11
- Python 3.9+ (для запуска `meshnet.py`) или готовый `meshnet.exe`
- Аккаунт на [cloudpub.ru](https://cloudpub.ru) (бесплатно)

## Установка

### 1. Скачайте репозиторий

```
git clone https://github.com/YOUR_USERNAME/AntiWhiteList
cd AntiWhiteList
```

Или скачайте ZIP и распакуйте в `C:\AntiWhiteList\`

### 2. Структура папок

После скачивания папка должна выглядеть так:

```
AntiWhiteList\
├── launcher.bat       <- запускать это
├── meshnet.py
├── xray.exe
├── clo.exe
├── config.json
├── tunnels.json
├── bin\               <- WinDivert и winws.exe
├── lists\             <- списки доменов
└── zapret\            <- батники шифратора
    ├── bin\
    ├── lists\
    ├── service.bat
    ├── general.bat
    └── general (ALT*).bat ...
```

### 3. Установите зависимости Python

```
pip install websockets psutil
```

### 4. Настройте CloudPub

1. Зарегистрируйтесь на [cloudpub.ru](https://cloudpub.ru)
2. Скачайте CLI-клиент (`clo.exe`) — положите в папку `AntiWhiteList\`
3. Авторизуйтесь:
   ```
   clo login
   ```
4. Зарегистрируйте три туннеля:
   ```
   clo register tcp 7700
   clo register tcp 1080
   clo register tcp 8443
   ```
5. Запустите все туннели:
   ```
   clo run
   ```
   CloudPub выдаст адреса вида `tcp.cloudpub.ru:XXXXX`

## Запуск

Запустите `launcher.bat` **от имени администратора** (UAC запросится автоматически).

```
Меню:
  1 - Start all
  2 - Check ports
  0 - Stop all and exit
```

При выборе `1` откроется список режимов zapret — выберите номер (начните с `1 - general`).

Порядок запуска:
1. zapret (обход DPI)
2. Xray (VLESS+Reality сервер на порту 8443)
3. CloudPub (`clo run` — поднимает туннели)
4. Meshnet (дашборд на `http://localhost:7701`)

---

## Настройка туннелей через дашборд

1. Откройте `http://localhost:7701`
2. Войдите: логин `meshnet`, пароль `mesh2026`
3. Введите адреса туннелей из CloudPub:

| Поле | Туннель для | Локальный порт |
|------|------------|----------------|
| P2P туннель | Meshnet P2P | 7700 |
| SOCKS5 туннель | Прокси для телефона по Wi-Fi | 1080 |
| Xray туннель | VLESS для мобильного интернета | 8443 |

4. Нажмите **Сохранить адреса**
5. Адреса сохранятся в `tunnels.json` и загрузятся автоматически при следующем запуске

## Настройка телефона

### Мобильный интернет — VLESS+Reality (через Xray)

Это работает даже когда Wi-Fi нет.

1. После заполнения поля **Xray туннель** в дашборде появится готовая VLESS-ссылка
2. Скопируйте её кнопкой **Скопировать VLESS**
3. Установите на телефон одно из приложений:
   - **Android**: [v2rayNG](https://github.com/2dust/v2rayNG/releases) или [Hiddify](https://hiddify.com)
   - **iOS**: Hiddify или Streisand
4. В приложении: `+` → **Импорт из буфера обмена** → подключить

### Wi-Fi — SOCKS5 прокси (через Meshnet)

1. В дашборде нажмите кнопку **Инструкция** в блоке прокси
2. Введите полученные IP и порт:

**Android:**
Настройки → Wi-Fi → долгое нажатие на сеть → Изменить → Прокси: Вручную → Сервер + Порт

**iOS:**
Настройки → Wi-Fi → (i) → Настроить прокси → Вручную → Сервер + Порт

## Режимы запрета (zapret)

При запуске предлагается выбрать один из 18 режимов. Если не знаете какой — начните с `1` (general). Если не работает — пробуйте по порядку.

| № | Режим | Описание |
|---|-------|----------|
| 1 | general | Стандартный, работает у большинства |
| 2–12 | ALT — ALT11 | Альтернативные варианты |
| 13–16 | FAKE TLS AUTO | Для операторов с глубокой инспекцией |
| 17–18 | SIMPLE FAKE | Лёгкий режим |

## Конфигурация Xray

Параметры VLESS+Reality в `config.json`:

```json
{
  "inbounds": [{
    "port": 8443,
    "protocol": "vless",
    "streamSettings": {
      "security": "reality",
      "realitySettings": {
        "dest": "vk.com:443",
        "serverNames": ["vk.com"],
        "privateKey": "...",
        "fingerprint": "chrome"
      }
    }
  }]
}
```

> Приватный ключ уникален для каждой установки. Публичный ключ (`pbk`) автоматически подставляется в VLESS-ссылку.

## Устранение неполадок

**Лаунчер закрывается сразу**
→ Запустите правой кнопкой → «Запуск от имени администратора»

**`[3/4] clo.exe not found`**
→ Скачайте `clo.exe` с [cloudpub.ru](https://cloudpub.ru) и положите в папку проекта
→ Выполните `clo login`

**`[1/4] NOT FOUND: zapret\general.bat`**
→ Распакуйте архив с батниками в папку `zapret\`

**VLESS-ссылка не появляется в дашборде**
→ Убедитесь что заполнено поле **Xray туннель** (не P2P и не SOCKS5)
→ Нажмите Сохранить ещё раз

**Телефон подключается но сайты не открываются**
→ Убедитесь что `xray.exe` запущен: в «Проверке портов» должен быть `:8443`
→ Попробуйте другой режим zapret

**Порты заняты**
→ Выберите пункт `0` (Stop all) и запустите снова

## Безопасность

- Приватный ключ Xray хранится только в `config.json` на вашем ПК
- CloudPub передаёт трафик без расшифровки (TCP туннель)
- VLESS+Reality маскируется под легитимный HTTPS трафик к `vk.com`
- Меняйте пароли дашборда (`AUTH_LOGIN`, `AUTH_PASS` в `meshnet.py`) если используете в сети с чужими устройствами
