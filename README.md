# AntiWhiteList

Запускаете один `.bat` файл — и получаете работающий прокси для ПК и телефона. На ПК поднимается Xray (VLESS+Reality), zapret обходит DPI провайдера, CloudPub бесплатно пробрасывает порты в интернет. Веб-дашборд автоматически генерирует VLESS-ссылку для импорта в v2rayNG или Hiddify.

## Что внутри

| Компонент | Описание |
|-----------|----------|
| **Xray** (VLESS+Reality) | Прокси-сервер, маскируется под обычный HTTPS |
| **zapret** | Обход DPI провайдера |
| **CloudPub** (`clo.exe`) | Бесплатный TCP туннель — пробрасывает порты без настройки роутера |
| **Meshnet** | Веб-дашборд для управления туннелями и настройки телефона |

## Требования

- Windows 10 / 11
- Python 3.9+ — [python.org/downloads](https://www.python.org/downloads/)
- Аккаунт на [cloudpub.ru](https://cloudpub.ru) (бесплатно)

## Установка

### 1. Скачайте репозиторий

```
git clone https://github.com/Oimpr/AntiWhiteList
cd AntiWhiteList
```

Или скачайте ZIP и распакуйте в `C:\AntiWhiteList\`

### 2. Структура папок

После скачивания папка должна выглядеть так:

```
AntiWhiteList\
├── launcher.bat       <- запускать это
├── meshnet.py
├── xray.exe
├── clo.exe
├── config.json
├── tunnels.json
├── bin\               <- WinDivert и winws.exe
├── lists\             <- списки доменов
└── zapret\            <- батники шифратора
    ├── bin\
    ├── lists\
    ├── service.bat
    ├── general.bat
    └── general (ALT*).bat ...
```

### 3. Установите зависимости Python

Откройте PowerShell от имени администратора и выполните:

```powershell
pip install websockets psutil
```

Если `pip` не найден — при установке Python поставьте галочку **Add Python to PATH**, затем перезапустите PowerShell.

Проверить что всё установилось:

```powershell
python -c "import websockets, psutil; print('OK')"
```

### 4. Настройте CloudPub

1. Зарегистрируйтесь на [cloudpub.ru](https://cloudpub.ru)
2. Скачайте CLI-клиент (`clo.exe`) — положите в папку `AntiWhiteList\`
3. Авторизуйтесь в PowerShell:
   ```powershell
   .\clo.exe login
   ```
4. Зарегистрируйте три туннеля:
   ```powershell
   .\clo.exe register tcp 7700
   .\clo.exe register tcp 1080
   .\clo.exe register tcp 8443
   ```
5. Проверьте что туннели зарегистрированы:
   ```powershell
   .\clo.exe ls
   ```

## Запуск

Запустите `launcher.bat` — UAC запросит права администратора автоматически.

```
AntiWhiteList + Meshnet
----------------------------------------
  1 - Start all
  2 - Check ports
  0 - Stop all and exit
```

При выборе `1` откроется список режимов zapret — введите номер (начните с `1`).

Порядок запуска:
1. zapret (обход DPI)
2. Xray (VLESS+Reality сервер на порту 8443)
3. CloudPub (`clo run` — поднимает туннели)
4. Meshnet (дашборд на `http://localhost:7701`)

---

## Настройка туннелей через дашборд

1. Откройте `http://localhost:7701`
2. Войдите: логин `meshnet`, пароль `mesh2026`
3. Введите адреса туннелей из CloudPub (вида `tcp.cloudpub.ru:XXXXX`):

| Поле | Назначение | Порт |
|------|-----------|------|
| P2P туннель | Meshnet P2P соединение | 7700 |
| SOCKS5 туннель | Прокси для телефона по Wi-Fi | 1080 |
| Xray туннель | VLESS для мобильного интернета | 8443 |

4. Нажмите **Сохранить адреса**
5. Адреса сохранятся в `tunnels.json` и загрузятся автоматически при следующем запуске

---

## Настройка телефона

1. После заполнения поля **Xray туннель** в дашборде появится готовая VLESS-ссылка
2. Нажмите **Скопировать VLESS**
3. Установите приложение:
   - **Android**: [v2rayNG](https://github.com/2dust/v2rayNG/releases) или [Hiddify](https://hiddify.com)
   - **iOS**: Hiddify или Streisand
4. В приложении: `+` → **Импорт из буфера обмена** → подключить

### Wi-Fi — SOCKS5 прокси

1. В дашборде нажмите **Инструкция** в блоке прокси
2. Введите IP и порт из поля SOCKS5 туннель:

**Android:** Настройки → Wi-Fi → долгое нажатие на сеть → Изменить → Прокси: Вручную → Сервер + Порт

**iOS:** Настройки → Wi-Fi → (i) → Настроить прокси → Вручную → Сервер + Порт

## Режимы zapret

При запуске предлагается выбрать один из 18 режимов. Начните с `1`. Если не работает — пробуйте по порядку.

| № | Режим | Описание |
|---|-------|----------|
| 1 | general | Стандартный, работает у большинства |
| 2–12 | ALT — ALT11 | Альтернативные варианты |
| 13–16 | FAKE TLS AUTO | Для операторов с глубокой инспекцией пакетов |
| 17–18 | SIMPLE FAKE | Лёгкий режим |

## Конфигурация Xray

Параметры VLESS+Reality хранятся в `config.json`. Приватный ключ уникален для каждой установки и генерируется один раз. Публичный ключ (`pbk`) автоматически подставляется в VLESS-ссылку в дашборде.

Для смены ключей:

```powershell
.\xray.exe x25519
```

Скопируйте `Private key` в `config.json` → поле `privateKey`, а `Public key` — в дашборд или оставьте для ручной сборки ссылки.

## Устранение неполадок

**Лаунчер закрывается сразу**
→ Правый клик на `launcher.bat` → «Запуск от имени администратора»

**`[3/4] clo.exe not found`**
→ Скачайте `clo.exe` с [cloudpub.ru](https://cloudpub.ru), положите в папку проекта
→ Выполните `.\clo.exe login`

**`[1/4] NOT FOUND: zapret\general.bat`**
→ Распакуйте архив с батниками в папку `zapret\`

**VLESS-ссылка не появляется в дашборде**
→ Заполните именно поле **Xray туннель** (не P2P и не SOCKS5) и нажмите Сохранить

**Телефон подключается но сайты не открываются**
→ Убедитесь что `xray.exe` запущен: пункт `2 - Check ports` → должен быть `:8443`
→ Попробуйте другой режим zapret

**`python` не найден при запуске Meshnet**
→ Установите Python с [python.org](https://www.python.org/downloads/) с галочкой **Add to PATH**
→ Выполните `pip install websockets psutil`

## Безопасность

- Приватный ключ Xray хранится только в `config.json` на вашем ПК
- CloudPub передаёт трафик без расшифровки (TCP туннель)
- VLESS+Reality маскируется под легитимный HTTPS трафик к `vk.com`
- Смените пароли дашборда в `meshnet.py` (поля `AUTH_LOGIN`, `AUTH_PASS`) если используете в сети с чужими устройствами
